<result>**Task: Research mutmut incremental/cache mechanism**

**Storage**: `mutants/` directory (mutmut 3.x). Older versions used a SQLite file (`.mutmut-cache`).

**How resume works**: `mutmut run` automatically skips already-tested mutants. Re-runs only mutants in functions whose source code changed since last run.

**How to reset**: Delete `mutants/` — that's the only reset mechanism. There is **no `--rerun-all` flag** in mutmut; that flag doesn't exist. To selectively rerun survivors: `for ID in $(mutmut result-ids survived); do mutmut run $ID; done`.

**Implication for local-engineering.prompt**: The line "never pass `--rerun-all`" references a nonexistent flag. The real guard is "never delete `mutants/` between agent runs."
